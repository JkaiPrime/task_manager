## Deployment

To deploy this project run

```bash
  python -m venv venv
```
- Windows
```bash
  venv\Scripts\activate
```
- No Linux/MacOS:
```bash
  source venv/bin/activate
```
Next run:
```bash
  pip install -r requirements.txt
```
Last step:
```bash
  python app.py
```
